# Project Details

This project is proposed in accordance of ELE529E Lecture at Istanbul Technical University.  

## Abstract 
 
This project covers freertos, device drivers, camera and LCD[^1] interface. What this project, proposes is filtering incoming frames from Camera and then displaying it on LCD display using RTOS[^2], DMA[^3], Interrupts and task scheduling subject on STM32F429I Discovery Board which has built-in LCD on itself. 


[^1]: Liquid Crystal Display
[^2]: Real-Time Operating System
[^3]: Direct Memory Access
